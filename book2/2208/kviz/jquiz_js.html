Osnove JavaScript

Kviz

 
  1. Znotraj katerega elementa HTML vpišemo programsko kodo JavaScript?
    1.   <javascript>
    2.   <scripting>
    3.   <js>
    4.   <script>
  2. Kje v dokumentu HTML je najustreznejše mesto za vključitev programov JavaScript?
    1.   V razdelku <head>
    2.   V razdelku <body>
    3.   V razdelku <head> ali razdelku <body>
    4.   Na koncu dokumenta HTML, pred </body>
  3. S katero značko HTML vključimo zunanjo datoteko JavaScript?
    1.   <script href="mojakoda.js"></script>
    2.   <script name="mojakoda.js"></script>
    3.   <script src="mojakoda.js"></script>
  4. Zunanja datoteka JavaScript mora vsebovati značko <script>.
    1.   Pravilno
    2.   Nepravilno
  5. Kako dodamo komentar v program JavaScript?
    1.   ' To je komentar
    2.   <!-- To je komentar -->
    3.   // To je komentar
  6. Programski jezik JavaScript je enak kot programski jezik Java.
    1.   Pravilno
    2.   Nepravilno
  7. Kateri dogodek se zgodi, ko uporabnik klikne na element HTML?
    1.   onMouseClick
    2.   onChange
    3.   onClick
    4.   onMouseOver
  8. Kako napovemo spremenljivko v programskem jeziku JavaScript?
    1.   v imeOsebe;
    2.   var imeOsebe;
    3.   variable imeOsebe;
  9. Kateri operator uporabimo v programskem jeziku JavaScript za prirejanje vrednosti spremenljivki?
    1.   =
    2.   -
    3.   *
    4.   ==
  10. Ali programski jezik JavaScript razlikuje med velikimi in malimi črkami?
    1.   Ne
    2.   Da
