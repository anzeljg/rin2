50. The ttk element layer

A ttk element is one of the pieces that make up a widget. In order to understand how elements are assembled into styles, read these sections.

  • Section 50.1, “ttk layouts: Structuring a style”: the static structure of elements within a widget.

  • Section 50.2, “ttk style maps: dynamic appearance changes”: states of a widget and how those states affect its appearance.
