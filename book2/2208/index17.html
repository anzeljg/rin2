Osnove CSS

Slogi posameznega elementa

Pravila CSS lahko vstavimo neposredno v atribut style vsakega elementa HTML. Za ilustracijo oblikujmo nedelujočo povezavo na spletno stran tako, da bo povezava prikazana prečrtano. To storimo tako, da v element HTML dodamo atribut style, ki vsebuje pravilo CSS text-decoration:line-through.

Tako kot pri slogih, določenih za posamezno spletno stran, tudi tu uporabljamo isto sintakso CSS. Ker pa so vsa pravila CSS v enem atributu, jih je treba strniti v eno vrstico. Vgrajeni slogi so najbolj specifičen način določanja CSS. Pravila CSS, definirana v atributu posameznega elementa HTML, preglasijo vsa prejšnja pravila CSS v zvezi s tem elementom.



Vgrajenim slogom se moramo izogibati za vsako ceno, ker onemogočajo spreminjanje slogov iz ločene slogovne datoteke. Če bi kdaj želeli preoblikovati svoje spletišče, ne bi bilo dovolj le spremeniti nekaj pravil CSS v slogovni datoteki slogi.css – namesto tega bi morali pregledati vsako posamezno spletno stran in posodobiti vsak posamezen element HTML, ki vsebuje določen slogovni atribut. To bi bilo naravnost grozljivo.

Pravila CSS je mogoče razširiti na več zunanjih slogovnih datotek tako, da na spletno stran dodamo več elementov <link />. Pogost primer uporabe je ločevanje slogov za različne dele spletišča. To omogoča izbirno uporabo skladnih slogov za različne dele spletišča.
