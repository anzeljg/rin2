Naloge

10


(a) Izberi vse zapise, pri katerih je v stolpcu kraj črka »a« na drugem mestu. 

SELECT * FROM stranka WHERE kraj LIKE '_a%';

(b) Izberi vse zapise, pri katerih so v stolpcu kraj na prvem mestu črke »a«, »c« ali »s«. 

SELECT * FROM stranka WHERE kraj LIKE '[acs]%';

(c) Izberi vse zapise, pri katerih je v stolpcu kraj na prvem mestu črka med »a« in »g«. 

SELECT * FROM stranka WHERE kraj LIKE '[a-g]%';

(b) Izberi vse zapise, pri katerih v stolpcu kraj na prvem mestu nista črki »a« in »s«. 

SELECT * FROM stranka WHERE kraj LIKE '[!as]%';

11


(a) Uporabi operator IN ter izberi zapise, pri katerih drzava 'Avstrija' ali 'Francija'. 

SELECT * FROM stranka WHERE drzava IN ('Avstrija','Francija');

(b) Uporabi operator BETWEEN ter izberi izdelke, katerih cena je med 10 in 20. 

SELECT * FROM izdelek WHERE cena BETWEEN 10 AND 20;


12


(a) Ustvari novo podatkovno bazo test. 

CREATE DATABASE test;

(b) Izbriši podatkovno bazo test. 

DROP DATABASE test;

(c) Ustvari novo tabelo oseba. 

CREATE TABLE oseba (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  ime TEXT NOT NULL,
  priimek TEXT NOT NULL,
  naslov TEXT,
  mesto TEXT
);

(d) Dodaj stolpec rojstni_dan, ki vsebuje vrsto podatkov DATE, tabeli oseba. 

ALTER TABLE oseba
ADD rojstni_dan DATE;

(e) Izbriši stolpec rojstni_dan iz tabele oseba. 

ALTER TABLE oseba
DROP COLUMN rojstni_dan;

(f) Izbriši tabelo oseba. 

DROP TABLE oseba;
