Igra Black Jack

Oglejmo si rokovalnike vseh treh gumbov, ki jih bomo uporabljali v igri.

Rokovalnik gumba »Deli« mora končati igro, če uporabnik gumb klikne medtem, ko igra poteka. Rokovalnik mora nastaviti ustrezni izid in povečati skupno število odigranih iger. Nato nastavi nov komplet igralnih kart in ga premeša. Ustvari igralca in delivca ter vsakemu izmed njiju izmenično razdeli dve karti. Nazadnje nastavi še spremenljivko, ki beleži stanje igre – ali igra poteka, ali ne.

def deli():
  # Če kliknemo gumb medtem, ko igra poteka
  global izid, REZULTAT, igra_poteka
  if igra_poteka:
    izid = "Jack je zmagal."
    REZULTAT["SKUPAJ"] += 1

  global komplet
  komplet = Komplet()
  komplet.premesaj()

  global igralec, delivec
  igralec = Igralec()
  delivec = Igralec(True)
  igralec.dodaj_karto(komplet.deli_karto())
  delivec.dodaj_karto(komplet.deli_karto())
  igralec.dodaj_karto(komplet.deli_karto())
  delivec.dodaj_karto(komplet.deli_karto())

  izid = "Na vrsti si. Vleci ali obdrži?"
  igra_poteka = True

Rokovalnik gumba »Vleci« igralcu doda novo karto. Nato preveri, ali skupna vrednost kart igralca presega 21 in v tem primeru konča igro.


def vleci():
  # Če igra poteka, potem igralec vleče karto
  global izid, REZULTAT, igra_poteka, komplet, igralec
  if igra_poteka:
    igralec.dodaj_karto(komplet.deli_karto())
  # Če igralec preseže 21, potem končaj igro
  if igralec.vrednost_kart() > 21:
    izid = "Imaš več kot 21. Jack je zmagal. Nova igra?"
    igra_poteka = False
    REZULTAT["SKUPAJ"] += 1

Rokovalnik gumba »Obdrži« začne delivcu dodajati nove karte, dokler je njihova skupna vrednost manjša od 17. Nato preveri, ali skupna vrednost kart delivca presega 21 – v tem primeru je igralec zmagovalec igre. V nasprotnem primeru pa preveri, kdo ima višjo skupno vrednost kart ter tako določi zmagovalca.

def obdrzi():
  # Če igra poteka, potem delivec vleče karte, dokler
  global izid, REZULTAT, igra_poteka, komplet, delivec
  while delivec.vrednost_kart() < 17:
    delivec.dodaj_karto(komplet.deli_karto())
  igra_poteka = False
  # Končaj igro in določi zmagovalca
  if delivec.vrednost_kart() > 21:
    izid = "Jack ima več kot 21. Zmagal si! Nova igra?"
    REZULTAT["SKUPAJ"] += 1
    REZULTAT["ZMAGE"] += 1
  else:
    REZULTAT["SKUPAJ"] += 1
    if igralec.vrednost_kart() > delivec.vrednost_kart():
      izid = "Zmagal si! Nova igra?"
      REZULTAT["ZMAGE"] += 1
    else:
      izid = "Jack je zmagal. Nova igra?"
