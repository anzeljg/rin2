class Tocka:
  # ...

  def razdalja_do(self, tocka):
    """ Vrne razdaljo do druge točke """
    dx = self._x - tocka._x
    dy = self._y - tocka._y
    return (dx ** 2 + dy ** 2) ** 0.5

Rešitev

class Tocka:
  # ...

  def premica_skozi(self, tocka):
    """ Vrne koeficienta premice skozi dani točki """
    k = (self._y - tocka._y) / (self._x - tocka._x)
    n = self._y - k * self._x
    return "(" + str(k) + ", " + str(n) + ")"

Metoda ne bo delovala, kadar bosta $x$ koordinati obeh točk enaki. V tem primeru bi prišlo do deljenja s številom 0, kar pa ni definirano.

Načrtovanje

  1. načrtaj 3 dane točke: $A$, $B$ in $C$
  2. načrtaj daljico $\overline{AB}$
  3. načrtaj daljico $\overline{BC}$
  4. načrtaj premico $p$, ki razpolavlja daljico $\overline{AB}$ v točki $X$
  5. načrtaj premico $q$, ki razpolavlja daljico $\overline{BC}$ v točki $Y$
  6. načrtaj krožnico s središčem v točki $S$ (presečišče daljic $p$ in $q$) in polmerom $r = \overline{AS}$

Načrtovanje

Računanje koordinat središča

Če imamo dane tri točke $T_{1}(x_{1}, y_{1})$, $T_{2}(x_{2}, y_{2})$ in $T_{3}(x_{3}, y_{3})$, lahko ustvarimo dve premici tako, da premica $a$ poteka skozi točki $T_{1}$ in $T_{2}$, premica $b$ pa skozi točki $T_{2}$ in $T_{3}$.

Enačbi teh dveh premic sta:

$$y_{a} = n_{a} (x - x_{1}) + y_{1}$$ $$y_{b} = n_{b} (x - x_{2}) + y_{2}$$

kjer $n$ predstavlja smerni koeficient oziroma naklon, ki ga izračunamo tako:

$$n_{a} = \frac{y_{2} - y_{1}}{x_{2} - y_{1}}$$ $$n_{b} = \frac{y_{3} - y_{2}}{x_{3} - y_{2}}$$

Središče krožnice je presečišče premic, ki sta pravokotni na daljici $\overline{T_{1}T_{2}}$ in $\overline{T_{2}T_{3}}$ ter hkrati potekata skozi središči obeh daljic. Pravokotnica na premico s smernim koeficinetom oziroma naklonom $n$ ima smerni koeficient oziroma naklon $-\frac{1}{n}$. Zato sta enačbi premic, ki sta pravokotni na premici $a$ in $b$ ter potekata skozi središči daljic $\overline{T_{1}T_{2}}$ in $\overline{T_{2}T_{3}}$, enaki:

$$y'_{a} = -\frac{1}{n_{a}}\left ( x - \frac{x_{1} + x_{2}}{2} \right ) + \frac{y_{1} + y_{2}}{2}$$ $$y'_{b} = -\frac{1}{n_{b}}\left ( x - \frac{x_{2} + x_{3}}{2} \right ) + \frac{y_{2} + y_{3}}{2}$$

Ker se ti dve premici sekata v središču krožnice, lahko iz njunih enačb izračunamo koordinato $x$ središča krožnice:

$$x = \frac{n_{a}n_{b}(y_{1} - y_{3}) + n_{b}(x_{1} + x_{2}) - n_{a}(x_{2} + x_{3})}{2(n_{b} - n_{a})}$$

Koordinato $y$ središča krožnice pa lahko izračunamo tako, da $x$ vstavimo v eno izmed gornjih enačb pravokotnic.

Računanje polmera

Ko smo izračunali koordinati središča krožnice $S(x, y)$ je računanje dolžine polmera $r$ precej enostavno. Izračunati moramo samo razdaljo med središčem $S$ in točko, ki leži na krožnici, na primer $T_{1}$.

$$r = \sqrt{(x - x_{1})^{2} + (y - y_{1})^{2}}$$

Podrobna razlaga je na voljo tukaj: Equation of a Circle from 3 Points (2 dimensions)

Rešitev

def sredisce_kroznice(a, b, c):
  """ Vrne središče krožnice skozi tri dane točke """
  s = Tocka()
  na = (b.y() - a.y()) / (b.x() - a.x()) # naklon premice a
  nb = (c.y() - b.y()) / (c.x() - b.x()) # naklon premice b
  # imenovalec ulomka za izračun koordinate x
  im = na*nb*(a.y() - c.y()) + nb*(a.x() + b.x()) - na*(b.x() + c.x())
  # števec ulomka za izračun koordinate x
  st = 2*(nb - na)
  # izračun koordinate x središča krožnice
  s.x(im / st)
  # izračun koordinate y središča krožnice
  s.y(-1*(s.x() - (a.x() + b.x())/2)/na + (a.y() + b.y())/2)

  return s

def polmer_kroznice(a, s):
  """ Vrne polmer krožnice skozi tri dane točke """
  return ((s.x() - a.x()) ** 2 + (s.y() - a.y()) ** 2) ** 0.5

Naloge

4

Razredu Tocka dodaj metodo razdalja_do, ki vrne razdaljo med dvema točkama.
Na primer Tocka(3, 5).razdalja_do(Tocka(-1, 2)) vrne 5.0.

5

Enačba premice je $y = kx + n$. Koeficienta $k$ in $n$ v celoti opisujeta premico. Razredu Tocka dodaj metodo premica_skozi, ki bo izračunala premico skozi dani točki. Metoda naj vrne oba koeficienta, kot dvojico vrednosti $(k, n)$.
Na primer Tocka(4, 11).premica_skozi(Tocka(6, 15)) vrne (2.0, 3.0).

Kdaj ta metoda ne bo delovala?

6

Napiši funkcijo, ki izračuna koordinati središča krožnice in polmer krožnice, če imaš podane tri točke, ki ležijo na tej krožnici.

Primer: Če imaš podane točke $A = (1, 1)$, $B = (2, 4)$ in $C = (5, 3)$, potem je središče krožnice točka $S = (3, 2)$ in polmer krožnice $r = \sqrt{5}$.
